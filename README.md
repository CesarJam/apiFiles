# apiFiles  
API backend para el desarrollo de Archivo de Gestión Documental

## Instalar

Para instalar las dependencias principales:
```bash
npm i express cors
```

instalar dependencias dentro de functions
```bash
cd functions
npm install
```
Regresa a la raíz de tu proyecto y prueba
```bash
firebase serve
```
